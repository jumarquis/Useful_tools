** Delete local branch
$ git branch -D <branch_name>

** Delete remote branch
$ git push origin --delete <branch_name>

** Déplacer une branche (A) sur une autre (B)
$ git checkout A 
$ git rebase A~ --onto B 

** Modification du dernier commit
$ git add 
plus
$ git commit --amend            -> modifier dernier commit et nom
$ git commit --ament --no-edit  -> le contenu seulement
